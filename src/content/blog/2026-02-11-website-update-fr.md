# Mise a jour du site : Plus rapide, Plus intelligent, Mobile-First

On a bosse dur pour ameliorer le site WITM, et aujourd'hui on est contents de partager les resultats !

## Quoi de neuf

### Navigation Mobile-First

L'experience mobile a ete entierement repensee. Le menu de navigation est maintenant un overlay plein ecran avec de grandes zones tactiles et des animations fluides. Fini les panneaux lateraux bancals — juste un beau menu centre qui marche parfaitement sur tous les telephones.

### Boost de Performance

On a optimise le chargement du site :

- **Bundle initial 68% plus petit** — le fichier JavaScript principal est passe de 825KB a 266KB grace au code splitting.
- **Lazy loading des images** — les photos et captures d'ecran ne se chargent que quand vous scrollez dessus.
- **Cache d'1 an** sur les fichiers statiques — les visiteurs qui reviennent ont un chargement instantane.
- **Compression Brotli + HTTP/3** active via Cloudflare.

### SEO & Visibilite

- **Sitemap** complet avec support multi-langues (anglais, francais, allemand) et balises hreflang.
- **Donnees structurees** (JSON-LD) pour que Google comprenne mieux notre contenu.
- Balises **Open Graph** et **Twitter Card** pour de beaux aperçus de liens.

### Integration Cloudflare

On a migre notre DNS et securite vers **Cloudflare**, ce qui apporte :

- **CDN mondial** — le site est servi depuis le datacenter le plus proche de vous.
- **Protection DDoS** et WAF gere.
- **HSTS** et en-tetes de securite pour une navigation plus sure.
- **Early Hints + 0-RTT** pour des connexions TLS plus rapides.
